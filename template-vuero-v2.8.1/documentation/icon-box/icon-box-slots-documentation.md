### اسلات (Slots) VButton

اسلات های کامل موجود برای کامپوننت `<VButton />` در اینجا آمده است:

| اسلات                      | ویژگی                               | توضیحات                |
| -------------------------- | ----------------------------------- | ---------------------- |
| #default<br />**required** | <span class="is-null">`null`</span> | آیکون یا تصویر به باکس |
