### پراپس VMessage

در اینجا پراپس کامل موجود برای کامپوننت `<VMessage />` آمده است:

| پراپس    | پیش فرض                                       | نوع                                                        |
| -------- | --------------------------------------------- | ---------------------------------------------------------- |
| color    | <span class="is-undefined">`undefined`</span> | `primary`, `success`, `info`, `warning`, `danger`, `white` |
| closable | <span class="is-boolean">`false`</span>       | boolean                                                    |
