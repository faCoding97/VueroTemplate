### رویداد های VMessage

رویدادهای کامل موجود برای کامپوننت `<VMessage />` در اینجا آمده است:

| رویداد | کال بک (Callback)                             | توضیحات                                              |
| ------ | --------------------------------------------- | ---------------------------------------------------- |
| @close | <span class="is-function">`() => void`</span> | زمانی که کاربر روی آیکون بستن کلیک میکند اجرا میشود. |
