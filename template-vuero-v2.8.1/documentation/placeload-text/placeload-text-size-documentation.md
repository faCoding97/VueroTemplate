### اندازه

اندازه کامپوننت `<VPlaceloadText />` را برای مطابقت با رابط کاربری خود کنترل کنید.
می‌توانید ویژگی‌های `height` و `width` را بر اساس نیاز خود تنظیم کنید

<!--code-->

```vue
<template>
  <VPlaceloadText :lines="5" width="75%" last-line-width="25%" />
</template>
```
