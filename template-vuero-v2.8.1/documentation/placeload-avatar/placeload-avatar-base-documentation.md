### VPlaceloadAvatar

Vuero یک کامپوننت `<VPlaceloadAvatar />` برای نمایش اسکلتون آواتار در حین بارگذاری ارائه می‌کند. برای جزئیات بیشتر نشانه گذاری را بررسی کنید.

<!--code-->

```vue
<template>
  <VPlaceloadAvatar />
</template>
```
